<h1>Web Application Development</h1>
<h2>Course Code: ITM 1131</h2>
<h2>Course Level: Year One, Semester One</h2>
<h2>Credit Unit: 4</h2>
<h3>Course Description</h3>
<p>This course introduces students to web technologies-HTML, CSS, JavaScript;CSS framework; students will learn to implement modern web pages using HTML and CSS frameworks. The course also introduces students to full stack webapplication programming using JavaScript.</p>
<h3>Course Objectives</h3>
<ol>
<li>Describe Web Technologies(HTML, CSS & JavaScript), CSS framework, and javaScrip frameworks.</li>
<li>Implement a simple but modern web page withHTML and CSS.</li>
<li>Build a simple but functional web application using JavaScript.</li>
</ol>
<h3>Learning Outcomes</h3>
<ul>
<li>Use wb technologies(HTML, CSS & JavaScript),CSS framework, and javaScrip frameworks.</li>
<li>Design and create a simple but functional webpages using HTML and CSS frameworks.</li>
<li>Compose and build a simple full stackweb application using JavaScript framework.</li>
</ul>
